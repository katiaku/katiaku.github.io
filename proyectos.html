    Simon Game App

    Juego de memória contra el ordenador con el objetivo de memorizar una sequencia de colores cada vez más larga. Con el botón Start empezamos el huego y pulsando sobre los botones de colores intentamos repetir la secuencia mostrada por el ordenador. Comprobamos con el botón Process y subimos un nivel con el Continue. Existe la posibilidad de resetear el juego.

    GitHub www.simon.com
